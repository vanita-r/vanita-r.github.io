Website: Kyle Beland's Design Firm Website

Website Checklist Evaluation

  • 1. Submission Link: Does the link lead directly to the page to be reviewed? Yes
  • 2. File/Folder Names: Are there no spaces or uppercase letters in file/folder names (including scripts, images, etc.)? Yes
  • 3. Design:
    • a. Page has sufficient contrast and font sizing for readability. Yes, the color contrast and font sizing are good for readability.
    • b. Page uses site colors/fonts via a standard .css file. Yes
    • c. CRAP Principles:
      • i. Contrast - No. The black font is easy to read on the beigish backgorund. However, the blue font may be an issue.
      • ii. Repetition - Yes, the design elements are repeated consistently.
      • iii. Alignment - Yes, the elements are aligned properly.
      • iv. Proximity - Yes, related elements are grouped together.
  • 4. Page Structure: Does the page contain a Header, Main, and Footer? Yes
  • 5. Header: Is the brand/site header text in an <h1>? Yes
    • a. If this is a homework page it's likely something like Student Name * Course ID Course Name. No
    • b. If this is a "branded page" it should be the brand/company. Yes
    • c. DO NOT include page name in header. No, the page name is included in the header.
  • 6. Main Content: Does the main section start with the page name in an <h2>? (Should not include site/brand name). No, the h2 is not in the main section.
  • 7. Brand Tagline: Is there a consistent brand tagline or slogan on every page? Yes
  • 8. Footer:
    • a. Includes a menu for the user's pages. No
    • b. Includes HTML validation button/link. No
    • c. Includes CSS validation button/link. No

Constructive Feedback

  • The Good: The website looks very modern and visually appealing.
  • The Bad: The page name is included in the header, which goes against the best practice of not including the page name in the header.
  • The Ugly: The blue font may be an issue for readability against the beige background. So, I would make the blue font darker or use a different color that provides better contrast.
